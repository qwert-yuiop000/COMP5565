# COMP5565 - Decentralized Product Provenance System

This repository contains the source code for a blockchain-based decentralized product provenance and warranty management system.

## Repository Structure
- **5565.sol** - Backend smart contract (Solidity)
- **index.txt** - Frontend application code

## Frontend Installation
For the  frontend and framework install, please visit:
[https://github.com/ALisstry/5565-front-end](https://github.com/ALisstry/5565-front-end)

## System Overview

### Roles
The system supports four distinct roles:

- **Manufacturer**: Registers products and transfers them to retailers
- **Retailer**: Sells products to consumers  
- **Customer**: Owns products and submits warranty claims
- **ServiceCenter**: Handles warranty claims

### Core Features
- Product registration and tracking
- Ownership transfer history
- Warranty management system
- Claim processing workflow
- Decentralized provenance tracking

## 系统概述

### 角色说明
系统支持四种不同角色：

- **制造商**：注册产品并转移给零售商
- **零售商**：向消费者销售产品
- **消费者**：拥有产品并提交保修申请
- **服务中心**：处理保修索赔

### 核心功能
- 产品注册与追踪
- 所有权转移历史
- 保修管理系统
- 索赔处理流程
- 去中心化溯源追踪
