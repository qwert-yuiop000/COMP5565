# COMP5565
This is source code,if you want to use this project,you can go here for front install: https://github.com/ALisstry/5565-front-end


This is a blockchain-based decentralized product provenance and warranty management system with the following four roles:

Manufacturer: Registers products and transfers them to retailers

Retailer: Sells products to consumers

Customer: Owns products and submits warranty claims

ServiceCenter: Handles warranty claims

and some basic functions
